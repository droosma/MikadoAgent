## Break Description

After updating `PaymentController` to use `IPaymentService` (node 1), `PaymentDTO`
is still in the `Controllers.Models` namespace. The new `IPaymentService` interface
expects it in `Payment.Contracts`, causing CS0246 errors.

## Error Output

```text
src/Payment/IPaymentService.cs(8,35): error CS0246: The type or namespace name 'PaymentDTO' could not be found
```

## Proposed Fix

Move `PaymentDTO` class from `Controllers.Models` namespace to `Payment.Contracts`
namespace. Update the `using` statements in all files that reference `PaymentDTO`.
