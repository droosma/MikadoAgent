## Break Description

After updating `PaymentController` to use `IPaymentService` (node 1),
`PaymentValidator` still references the concrete `PaymentProcessor` class directly
in its constructor, causing CS0246 (type not found) after `PaymentProcessor` was
moved into `PaymentService`.

## Error Output

```text
src/Payment/PaymentValidator.cs(47,12): error CS0246: The type or namespace name 'PaymentProcessor' could not be found
```

## Proposed Fix

Change `PaymentValidator` constructor to accept `IPaymentService` instead of
`PaymentProcessor`. Update the validation call on line 52 from
`_processor.Validate(amount)` to `_service.ValidatePayment(amount)`.
